# Evaluation artifacts

- `no-cost/` is the checked-in deterministic fixture report. It records zero live calls and zero spend.
- `load-full.json` and `load-quick.json` are local SQLite scale profiles with their own machine/profile limitations.
- Live ledgers/runs, latency markers, public-dataset conversions, manual transcripts, and competitor reports are intentionally not checked in. They may contain licensed or user-supplied material and are ignored by Git.

No public-dataset or ChatGPT/Codex result is present unless a human runs the documented import/capture protocol and retains the output locally.
