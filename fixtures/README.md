# Fixtures

Fixtures are deterministic, contain no personal data or credentials, and must not perform live provider calls. The browser demo, provider recordings, evaluation histories, and load generators may use different representations, but each fixture records its version, seed or checksum, and limitations.

- `demo-vault/` documents the immutable provenance demonstration.
- `public-datasets/` contains synthetic LongMemEval/HaluMem field-layout examples and checksums, never upstream records.
- `competitors/` contains a visible black-box protocol and blank capture template, never claimed product results.
- InfiniteBuild is generated by `packages/evaluation/src/infinite-build.ts`.
- SQLite scale fixtures are generated into temporary directories by `scripts/load-test.ts`.
